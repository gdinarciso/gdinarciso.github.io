CS50 - Web Programming

On terminal, type:


git clone "url of the repository"
download the repository from GitHub to computer

touch "filename"
creates a new file

git add "filename"
files to be commit next time in order the keep track them on GitHub

git commit -m "message"
get a "snapshot" of current repository and backup it
the flag "message" means a brief history of what it was just made

git commit -am "filename"
execute add and commit at the same time

git status

git push
upload the files and changes to GitHub
git push --set-upstream -origin "branchname"

git pull
download the files and changes on them
if someone else make changes the pull command will show each one
up to the user decide each maintain

git log

git reset
--hard "commit" bring the file to the "commit" stage
--hard origin/master bring the file back where got it from

git branch
list the branches inside the current repository
git branch "name"
start a new branch in order to bring the project to a new way

git checkout "branchname"
move from current branch to "branchname"

git merge "branchname"
merge the branch "branchname" to the current branch
